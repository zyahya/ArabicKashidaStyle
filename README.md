# Arabic Kashida Style

هذة الحزمة البسيطة تطبّق نمط الكشيدة أو التطويل على النص العربي، كالتالي:

**قبل التكشيد:**

السلام عليكم.

**بعد التكشيد:**

الـسـلـام عـلـيـكـم.

هذة الحزمة تحتوي فقط على دالتان، كما في المثال التالي:

`Apply(string)`:

```cs
var input = Kashida.Apply("السلام عليكم.");
Console.WriteLine(input); // الـسـلـام عـلـيـكـم.
```

`Clear(string)`:

```cs
var input = Kashida.Clear("الـسـلـام عـلـيـكـم.");
Console.WriteLine(input); // السلام عليكم.
```

> تنبية: هذة الحزمة حالياً تحت التطوير ولا تدعم الحروف المشكلة.
